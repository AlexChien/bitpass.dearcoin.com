About Bitpass

Secure Bitcoin-based Authentication System

Brief Introduction

Bitpass is a Bitcoin-based authentication protocol. The server provides an authentication token to the client, which the client signs with its private key. The server authenticates the client based on the signed token.

What can be done?

Bitpass can be used for website user authentication, with Bitcoin addresses being used as user IDs. The server never stores user passwords or other information, so user data is protected even if the server is hacked.
